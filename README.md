## GA4 E-commerce Funnel Analysis

Project Overview

This project analyzes user behavior in a GA4 e-commerce dataset using BigQuery and Tableau.

The goal of the analysis is to understand how users move through the purchase funnel, identify conversion bottlenecks, and evaluate user behavior across different traffic sources, devices, operating systems, and languages.


## Business Questions

* How many sessions were generated during the analyzed period?
* How many orders were completed?
* What is the conversion rate between funnel stages?
* Which traffic sources generate the most sessions?
* Which device categories are used most frequently?
* Which operating systems are most popular among users?
* Which device languages are most common?


## Tools Used

* Google Analytics 4 (GA4)
* BigQuery
* SQL
* Tableau Public


## Funnel Stages

The funnel consists of the following events:

1. session_start
2. view_item
3. add_to_cart
4. begin_checkout
5. purchase


## Key Metrics

* Total Sessions: 354,857
* Total Orders: 4,745
